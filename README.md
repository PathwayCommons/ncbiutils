# ncbiutils

Making retrieval of records from NCBI E-Utilities simple and robust.

## Installation

```bash
$ pip install ncbiutils
```

## Usage

`ncbiutils` can be used to as follows:

```python
from ncbiutils.ncbiutils import Medline

...
```
